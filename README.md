# VK Teams Notification Bot

Чат-бот для автоматической рассылки персонализированных уведомлений сотрудникам через корпоративное приложение VK WorkSpace (VK Teams).

---
## Возможности

### 👤 Для сотрудников

| Команда / Действие | Описание |
|-------------------|----------|
| `/start` | Запуск онбординга: выбор роли, отдела, пола. Персонализация приветствия |
| `/info` | Показать список доступных команд для сотрудника |
| `/profile` | Просмотр личных данных: имя, роль, отдел, пол, дата присоединения |
| `/change_dept` | Изменить свой отдел через интерактивные кнопки |
| Кнопка «Прочитано» | Отметка уведомления как прочитанного (влияет на статистику) |

### 🔑 Для администраторов

| Команда / Действие | Описание                                                       |
|-------------------|----------------------------------------------------------------|
| `/start` | Онбординг + доступ к админ-функциям                            |
| `/info` | Показать список команд администратора                          |
| `/event` | Cоздания уведомления (пошаговый сценарий)                      |
| `/my_events` | Просмотр списка всех созданных админом уведомлений с деталями  |
| `/report` | Статистика по рассылкам: доставлено, прочитано, процент открытий |

## Быстрый старт

### Требования
- Go 1.21+
- PostgreSQL 14+
- Токен бота из админ-панели VK WorkSpace

### Установка

1. Клонируйте репозиторий:
```bash
    git clone <repository-url>
    cd vkbot
````

2. Создайте файл конфигурации .env и заполните его данными (токены)
3. Установите зависимости
```bash
    go mod tidy
 ```
4. Запустите бота:
```bash
    # Для разработки
    go run cmd/bot/main.go

    # Для продакшена (сборка бинарника)
    go build -o vkbot cmd/bot/main.go
    ./vkbot
 ```
